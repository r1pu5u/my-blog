# r1pu5u.github.io

Personal portfolio and blog built with [Astro](https://astro.build). Deployed automatically to GitHub Pages at https://r1pu5u.github.io.

## Features

- ⚡ Static site generation with Astro 6
- 📝 Markdown blog with content collections and MDX
- 🏷️ Tag filtering, search, and date-based routing
- 🎨 Cyberpunk OS terminal aesthetic
- 📱 Fully responsive

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
